## 🌟 Overview

This is my **personal developer portfolio**, built to showcase who I am as a Full Stack Developer — my skills, real-world projects, professional services, and journey in web development. Designed with a strong focus on **clean UI, smooth animations, and performance**, this portfolio serves as a central hub for anyone looking to explore my work or get in touch.

---

## ✨ Key Features

- 🎨 **Modern, Responsive Design** — Looks great on desktop, tablet, and mobile
- 🧑‍💻 **Dynamic Project Showcase** — Highlights real, deployed full-stack projects with live links
- 🛠️ **Skills & Tech Stack Section** — Visual breakdown of languages, frameworks, and tools
- 💼 **Services Section** — Outlines the type of development work and collaboration I offer
- 📄 **Experience Timeline** — Learning journey and professional growth at a glance
- 📬 **Integrated Contact Form** — Direct way for visitors/recruiters to reach out
- ⚡ **Smooth Animations** — Polished transitions and micro-interactions for better UX
- 🌙 **Fast & SEO Optimized** — Built with Next.js for performance and search visibility

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/eyasinarafat3485-gif/portfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
